# summary

# impression
要は適材適所でReactなどのフレームワークを使った方が良いケースとそうでないケースがある。

jQueryはそろそろやめて、アプリやシステム構築の際はReactを使っていきたい。

「20 vanilla web JavaScript projects」が結構良さそうだったので、写経してみようか。
