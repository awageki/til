# summary
- nullの代わりにnilを使う（ここで一文字節約できる）
- メリット：サードパーティ製のツールを使わなくてもできることが多い
- GoはGitをサポートしているので、モジュールをインポートしたい場合、gitリポジトリ名を入力するだけで良い
```go
import (
    "github.com/lib/pq"
)
```
- どのOSにも対応可能
